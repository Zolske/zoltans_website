Operators

Spread Operator

The "spread operator" allows you to spread out elements of an iterable object such as an array, map, or set.

syntax rules

  • The "spread operator" consists of ... which are placed in front of the iterable object.
  • NOTE: there is also the "rest operator" which also consists of ... but it is used as "function parameters".

JavaScript spread operator and array manipulation

syntax description
1) Constructing array literal 
let ARRAY_2 = ['A', ...ARRAY_1, 'B'];
example 
let initialChars = ['C', 'D'];
let chars = [ 'C', ...initialChars, 'D'];
console.log(chars); // ["C", "A", "B", "D"]
  • The spread operator allows you to insert another array into the initialized array when you construct an array using the literal form.
2) Concatenating arrays 
let ARRAY_3 = [...ARRAY_2, ...ARRAY_1];
example 
let numbers = [1, 2];
let moreNumbers = [3, 4];
let allNumbers = [...numbers, ...moreNumbers];
console.log(allNumbers); // [1, 2, 3, 4]
  • Use the spread operator to concatenate two or more arrays.
3) Copying an array 
let ARRAY_COPY = [...ARRAY_ORIGINAL]
example 
let scores = [80, 70, 90];
let copiedScores = [...scores];
console.log(copiedScores); // [80, 70, 90]
  • The spread operator can copy an array instance but only one level deep! Anything deeper (multidimensional array) is a not a copy but a reference which are "linked" and not independent copies.
    • example 
    let original = [80, 70, [12,33]];
let copiedOriginal = [...original];
original[1] = 'Z';           // one level deep is true copy not reference
original[2][0] = 'K';        // more than one level is not a copy but a reference
console.log(original);       // [ 80, 'Z',['K', 33 ]]
console.log(copiedOriginal); // [ 80,  70,['K', 33 ]]
// 70 is not changed but 12 is changed to 'K'

rest parameter

A rest parameter allows you to represent an indefinite number of arguments as an array.

syntax rules

  • is represented with ... followed by an arbitrary name
  • must be the last parameter in an function definition
  • if there is no argument passed for the rest parameter, than it is used as an empty array
